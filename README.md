## Running

First, run the development server:

```bash
npm run dev
```

## Installation

```bash
npm i
```

## Local launch

Open in browser [http://localhost:3000](http://localhost:3000)

## Online view

Website is also launched online [https://gameuppixel.vercel.app](https://gameuppixel.vercel.app)
